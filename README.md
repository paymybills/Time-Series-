# Time Series Analysis Practicals

This repository contains practical exercises for the Time Series Analysis course.

## Structure

*   **Practical 1**: US Population Analysis (Data Import, Trend Identification).
*   **Practical 2**: Nottem Dataset Analysis (Seasonality Identification).
*   **Practical 3**: Decomposition of Time Series (Decomposing `nottem` and `AirPassengers` data).
*   **Practical 4**: Manual Decomposition and Detrending of US Population Data.

## Usage

Each practical is contained in its own directory. You can run the R scripts using:

```bash
cd PracticalX
Rscript practicalX.r
```
